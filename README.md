# dotfiles

Dot, dot dot, dot-dot dot-dot dot-dot.

## Sync checklist

```sh
sh scripts/apps-export.sh
sh scripts/cline-export.sh
sh scripts/git-export.sh
BOSSMAN=<redacted> sh scripts/brew-export.sh
BOSSMAN=<redacted> sh scripts/fish-export.sh
```
